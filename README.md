## FastAPI Service

Minimal FastAPI app with a start script that uses Poetry for dependency management.

### Prerequisites

- Install Poetry (one of):
  - macOS (Homebrew): `brew install poetry`
  - Official installer: `curl -sSL https://install.python-poetry.org | python3 -`

### Setup and Run

1) From the project root, install dependencies:

```bash
poetry install
```

2) Ensure you're inside the project directory, then prepare the virtual environment:

```bash
poetry env activate
```

3) Activate the environment by sourcing the command generated by Poetry:

```bash
source $(poetry env info --path)/bin/activate
```

4) Start the server from the project root:

```bash
./scripts/start.sh
```
