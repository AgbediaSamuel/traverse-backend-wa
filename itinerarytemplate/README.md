# Visual Itinerary Template

This is a code bundle for Visual Itinerary Template. The original project is available at https://www.figma.com/design/z66wEIcPo8i2qSiGf5JbNN/Visual-Itinerary-Template.

## Running the code

Run `pnpm install` to install the dependencies.

Run `pnpm dev` to start the development server.

## Nginx Setup (for ngrok/local development)

If using nginx reverse proxy, see `../../traverse-backend-wa/README.md` for setup instructions. The nginx config is located at `../../traverse-backend-wa/infra/nginx-traverse.conf`.
